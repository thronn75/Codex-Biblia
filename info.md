# Codex Biblia

Codex Biblia is a study application designed to help users explore the Bible through structured learning.

The goal of the app is not simply to provide a quiz, but to help readers become more familiar with the content, structure, and themes of the biblical text.

## Features

- Book introductions summarizing major themes
- Quiz questions based on biblical narratives
- Expert questions for deeper study
- Verse references for each answer
- Multiple language support
- Offline functionality using local JSON content
- Minimal and distraction-free interface

## Languages

Codex Biblia currently supports:

- English
- German
- Serbian / Croatian (Latin script)
- Slovak

## Biblical Text Sources

The application references Bible translations that are in the public domain:

- Daničić–Karadžić translation
- King James Version (KJV)
- Luther Bible 1912

The app itself does not distribute modern copyrighted Bible translations.

## Purpose

Codex Biblia aims to encourage readers to:

- learn the structure of the Bible
- understand key events and teachings
- become more confident in discussing Scripture

The app is designed as a learning tool and companion for personal study.
